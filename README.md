# 🚀 CareerVerse – Smart Job Portal

CareerVerse is a modern **job portal web application** designed to help users discover job opportunities, explore hiring companies, and manage job applications through a clean and intuitive interface.

The platform simplifies the **job search experience** for students, fresh graduates, and professionals.

## 🌟 Features

* 🔐 User Authentication (Login & Signup)
* 🔍 Search and Explore Jobs
* 🏢 Top Companies Hiring Section
* 📂 Popular Job Categories
* 💼 Detailed Job Listings
* ⭐ Save Jobs for Later
* 📊 Track Applied Jobs
* 📱 Fully Responsive Design
* 🎨 Clean and Modern UI

## 🛠 Tech Stack

**Frontend**

* HTML5
* CSS3
* JavaScript

**Storage**

* Browser LocalStorage

## 💡 Key Modules

### 🔐 Authentication

Users can create accounts and log in to access the job portal.

### 💼 Job Listings

Browse available job openings with company details, location, salary, and required skills.

### ⭐ Saved & Applied Jobs

Users can save jobs and track their applications.

### 🏢 Top Companies

Displays companies actively hiring on the platform.
